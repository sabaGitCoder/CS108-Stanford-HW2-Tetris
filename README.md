# Tetris - Stanford CS108 HW2

> A complete, object-oriented implementation of the classic arcade game Tetris. 

This repository contains my solution for Homework 2 of Stanford University's **CS108: Object-Oriented Systems Design**. The goal of this project was to build a robust, bug-free Tetris game from scratch, focusing heavily on OOP principles, state management, and algorithmic thinking.

## 🎮 Features
* **Complete Game Mechanics:** Piece generation, translation, rotation, and dropping.
* **Row Clearing & Scoring:** Automatically detects and clears filled rows, updating the player's score.
* **Brain/AI Integration:** Includes an automated "Brain" module that calculates the optimal next move.
* **Object-Oriented Design:** Clean separation of logic between the board, pieces, and user interface.
